GIT

git clone :copy a whole repository from server

git add :add files to server

git commit -m "message":discribe the changes on the commit with a message

git status:print the current commit on the repository(if just want to print one of files, git status )

git push:push commit to server (-f origin master)

git pull:pull down commit from server

merge conflicts:from HEAD to equal signs is my changes; from equal signs to the greater signs is remote changes

git log:show history commit be made

git restart --hard (often be number):back to a commit before

git restart --hard origin/master:back to the version which downloaded from server

git commit -am "massage":merge git add and git commit

git reflog:list history reference to commit

branch

git branch:show all branch

git branch :add branch

git checkout :change current branch

git merge :merge the branch with current branch

git fetch:download the latest commit from the origin

git merge origin/master

fork:which commit wont reflect to server
